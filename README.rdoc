== JOBS789

This application is my project 1 at General Assembly. It is a Job advertisement application. The main page displays a list of jobs. Employers can create an account and log in to post jobs and see a list of applications received for those jobs. Jobseekers can log in to apply for a job they are interested in.

The application whilst in construction was running at https://blooming-plains-2243.herokuapp.com/    it may or may not still be active there.
